The user may specify classifications to a specific genre by adding text files in the "genres" folder.
The following types of files can be created
genre_artists.txt
genre_albums.txt
genre_genres.txt

where "genre" is a standard or custom genre. 
Some examples
1) If the user wants the Bee Gees and ABBA to be classified as "classical", then the user would create a file called "Classical_artists.txt" containing two lines which would read "Bee Gees" and "ABBA" respectively, without the quotations.
2) To create a custom genre "Altechnic" which would group Electronic, New Age, and Dance, then a file called "Altechnic_genres.txt" would be created, containing three lines listing those three genres.
3) To force specific albums to a specific genre (such as "Christmas"), then a file "Christmas_albums.txt" should be created, listing the target album titles.
